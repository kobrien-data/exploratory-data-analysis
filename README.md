# Exploratory Data Analysis of Horror Movies
This project is to do an exploratory data analysis project on horror movies. The dataset comes from:
> https://www.kaggle.com/datasets/PromptCloudHQ/imdb-horror-movie-dataset

## Tools Used
To do this data analysis, I am using python, pandas, and matplotlib.

## First Steps
The first step I took was to check for missing values in the data and fill them. For this dataset, I chose to fill the missing values with 'Not Rated', 'Unknown', and 0 for review_rating which is type float.
